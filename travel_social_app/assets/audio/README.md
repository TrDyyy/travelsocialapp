# Audio Assets

## Required Files

### ringtone.mp3
- **Purpose**: Nhạc chuông khi có cuộc gọi đến
- **Duration**: 3-10 giây
- **Format**: MP3
- **Notes**: Sẽ được phát lặp lại cho đến khi trả lời hoặc từ chối

### call_end.mp3
- **Purpose**: Âm thanh khi kết thúc cuộc gọi
- **Duration**: 1-2 giây
- **Format**: MP3

## Tải file mẫu

Bạn có thể tải file âm thanh miễn phí từ:
- https://pixabay.com/sound-effects/
- https://freesound.org/
- https://mixkit.co/free-sound-effects/

Hoặc dùng âm thanh mặc định của hệ thống.

## Cài đặt

1. Tải file MP3 về
2. Đổi tên thành `ringtone.mp3` và `call_end.mp3`
3. Copy vào thư mục này
4. Chạy `flutter clean` và `flutter pub get`
